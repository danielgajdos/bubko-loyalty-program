# Kebab na Kyjevskom - Loyalty Program

## Overview
Loyalty program for Kebab na Kyjevskom fast food restaurant. Customers earn stamps for each product purchased, and get the 11th product free.

## Key Differences from Bubko
- **Product-based tracking**: Each product type tracked separately (Kebab, Pizza, Burger, etc.)
- **11th product free**: Buy 10, get 11th free (instead of points-based)
- **Multi-product visits**: Admin can add multiple products in one scan
- **Branding**: Red/orange kebab theme instead of blue kids theme

## Deployment
- **Frontend**: kebabnakyjevskom.vercel.app
- **Backend**: Railway (separate service in same account)
- **Database**: Railway MySQL (separate instance)
